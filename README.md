## 🚀 Demo

[Ver sitio en vivo](https://skin-laser-beauty.vercel.app)

## ✨ Características

- Diseño moderno y responsive
- Optimizado para SEO
- Formulario de contacto
- Secciones: Hero, Servicios, Sobre Nosotros, Testimonios, Booking
- Animaciones suaves
- Modo oscuro/claro adaptable

## 🛠️ Tecnologías

- **Next.js 16** - Framework React
- **Tailwind CSS v4** - Estilos
- **Vercel** - Hosting y deploy
